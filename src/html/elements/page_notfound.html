Soc Errors Data not found

Use the soc-not-found class when you need to display a message that something is not available on the page. It will be automatically styled to be larger text centered to to page. A subtitle can be added withing a 'small' element inside the div. Also, an image of magritte's famous pipe will be added when you add the 'pipe' class, only add a good caption to it.

No records found
emmet: 
.soc-not-found{No records found}
Don't worryBe happy
emmet: 
.soc-not-found{Don't worry}>small{Be happy}

.soc-not-found.image{No records found}

.soc-not-found.pipe{Ceci n'est pas une erreur}
