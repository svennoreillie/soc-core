Soc Errors Data not found

Use the soc-not-found class when you need to display a message that something is not available on the page. It will be automatically styled to be larger text centered to to page. An image of magritte's famous pipe will be added when you add the 'pipe' class, only add a good caption to it.

No records found
emmet: 
.soc-not-found{No records found}
No records found
emmet: 
.soc-not-found.image{No records found}
Ceci n'est pas une erreur
emmet: 
.soc-not-found.pipe{Ceci n'est pas une erreur}
