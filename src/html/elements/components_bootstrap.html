Components Bootstrap examples

This framework is built on top of a bootstrap Sass port, this allows for all known bootstrap components and style elements to be used as well. Some of these might have been overridden by the Soc-core style but efforts were made to keep this limited as much as possible. For more information visit the bootstrap website
